# Platformadaxili ödənişsiz marketplace

Mizan pul qəbul etmir, saxlamır və tərəflərə köçürmür. Alıcı ilə satıcı əlaqə qurur, qiymət/çatdırılma/ödəniş üsulunu öz aralarında razılaşdırır və ödəniş platformadan kənarda baş verir.

## Platformada yoxdur

- checkout və kart məlumatı toplama
- payment capture və payment intent
- wallet, escrow və payout
- platformadaxili refund
- komissiya settlement-i
- “ödəniş tamamlandı” kimi təsdiqlənməyən status

## Platformada var

- məhsul kataloqu
- satıcıya məhsulla bağlı sorğu göndərmək
- satıcının əlaqə üstünlüklərini görmək
- sorğunu açıq, cavablandırılmış və bağlanmış kimi idarə etmək
- tərəflərə platformanın ödəniş tərəfi olmadığını aydın göstərmək

UI-da `Səbətə at`, `Checkout` və `Ödə` əvəzinə `Satıcı ilə əlaqə saxla` və `Sorğu göndər` işlədilir. Satıcı panelində `Sifarişlər` əvəzinə `Müştəri sorğuları` və `Razılaşmalar` terminləri işlədilir.

Platforma gələcəkdə yalnız elan, əlaqə və moderasiya üçün haqq tətbiq etsə belə, bu ayrıca məhsul qərarı və ödəniş arxitekturası tələb edir.
